# Icons Directory

Place your icon files here:

- `icon.png` - Light theme icon (256x256px recommended)
- `icon@2x.png` - Light theme high-res icon (512x512px recommended)
- `dark_icon.png` - Dark theme icon (256x256px recommended)  
- `dark_icon@2x.png` - Dark theme high-res icon (512x512px recommended)

All icons should be PNG format with transparent background.

Home Assistant will automatically use these local icons for the Cala integration.
